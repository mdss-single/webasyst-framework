Компонент ul.thumbs предназначен для «плиточного» представления некоторого списка. Реализация основана на свойстве display: inline-block;.

Компонент используется, как правило, для показа изображений (фотографий). Изображения, обернутые в ссылку (a img), по умолчанию имеют скругленные края (border-radius: 0.25rem). Если в контексте вашего приложения это не оправданно, уберите скругление путем добавления класса img.squared.

Фиксированные размеры блоков

Чтобы явно зафиксировать ширину блоков, используйте один из дополнительных классов для ul.thumbs: .li50px, .li100px, .li150px, .li200px, .li250px, .li300px, .li350px. Контент не будет выступать за пределы заданной ширины (в примере ниже изображения с размерами 300х300 автоматически сжимаются до 150х150 пикселей).
